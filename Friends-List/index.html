Friends List

Description

The goal of this project was to create a working friends list, as you would see in a game, or on Steam. It would be a small server-client app, where the client is the one requesting the server via TCP sockets to search for users, as well as add, remove, and invite them to games. The server must be able to store all of it's user data in a database. I used Postgres for this. The app currently only uses localhost, but it could easily be changed to do server/client. After reaching all my inital local goals I lost interest to take it any further. What I did make works though, and I learned a few things, so it was mostly a success.
