Multi-Agent Motion Planning

Description

This project is a basic framework for motion planning when there are multiple agents in an environment with varying obstacles. Each agent needs to navigate to its destination without running into obstacles or other agents. So far there is only support for a single agent.

Difficulties

The main difficulties I have had so far were mainly figuring out the best way to render things using my game engine, since this is the first project I am using my engine for.

Features

  • Single agent navigation
  • Supports any number of static cylindrical obstacles with varying sizes
  • Probabilistic roadmap (PRM) for possible paths
  • A* used to find the optimal path within the PRM
  • PRM is visualized by square nodes and lines connecting each node to its neighbors
  • Realtime rendering
  • 3D user controlled camera

Tools Used

  • My game engine Progression for rendering, window management, and matrix libraries
