Description

The goal of this assignment is to create a particle system to help with the following four simulations: A ball bouncing on a floor, a water fountain, a fire, and fireworks.

Difficulties

Code

Download zip file of code Link to Github

Simulations

  • Ball Bouncing off floor

Features

  • 3D implementation
  • 3D user controlled camera
  • Billboarding used for the particles
  • Normal mapping to make the billboards look like real spheres

Controls

  • W,A,S,D: Move the camera forward, backwards, left, and right
  • Mouse: Rotate the camera
  • P: Pause the simulation

Tools Used

  • OpenGL for the rendering
  • SDL2 for the window management and user interaction
  • GLM for the vectors and matrices
