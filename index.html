Spline Editor

The curves above are made of sets of points, connected by creating a Catmull-Rom spline. In draw mode, click to add points to the current selected spline. In edit mode, the control points become visible, and you can move them individually and delete points.

General Controls

  • S: Switch between draw mode and edit mode
  • C: Clear the canvas

Draw Mode Controls

  • Left Click: Add a new point to the end of the current selected spline
  • Spacebar: The next point added will be the first on a new spline

Edit Mode Controls

  • Left Click and Hold on a Point: Grab a point to move around
  • D: Delete the point that is currently being grabbed
