Image Processing

Description

This was the second homework assignment in this class. It is a command line tool that takes in an image, and can do various image processing features to it. The command line parsing code and image loading/saving was given to us, but we had to implement all of the filters ourselves.

Difficulties

The main difficulties I had were figuring out how some of the filters worked. The ones that gave me the most trouble were probably contrast, rotate, and fish eye lens. The contrast took me a tiny bit to think of how it would work, but it was easy to implement after that. Rotate was the exact opposite, since I kept having small problems with sampling backwards. The fish eye lens was difficult both ways. I had no idea how they were normally done, so I tried viewing it as an image (2D rectangle) being wrapped onto a sphere. To do the filter, I had to figure out the math to 'unwrap' a sphere using spherical coordinates. It works okay, but the the edges of the images are very distorted. It's expected, but probably more distorted than I think looks the best.

Blur (low): ./a.out -input sample/millcity.jpg -blur 5 -output results/blur5.jpg

Blur (high): ./a.out -input sample/millcity.jpg -blur 17 -output results/blur17.jpg

Sharpen (low): ./a.out -input sample/millcity.jpg -sharpen 5 -output results/sharpen5.jpg
