Grade Notifier

Description

I got tired of checking my grades all the time, so I made a little python script to check them for me. It logs into Moodle where the grades are posted, downloads that webpage, and parses the grades out. If the grades are different than the last time it checked, it will send me an email notifying me about which grade changed and how.

Go to Github page

Tools Used

  • SMTPlib for the emailing portion
  • Requests lib for the authenticating and webpage downloading
  • HTMLparser for parsing the page
  • Cron to check every 10 minutes
