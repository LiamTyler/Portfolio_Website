Volume Comparison Tool

Description

Vital's application features many volume algoritms. For example, if you are looking at a scan of the chest, you could run an algorithm to segment out the heart, since that's the part you are currently interested in viewing. For the developers that work on those algorithms, they want to know when they come out with a new version of the algorithm, how it differs from the old one. The tool they had to do this comparison was over a decade old, using OpenGL 1. I redid the whole thing to use OpenGL 4, have a user interface, support much larger volumes, and added features to further enhance the comparison.

Tools Used

  • OpenGL for the rendering
  • Nanogui for the windowing and UI

Features

  • Interactive volume comparison of volumes up to around 10GB in size
  • User interface for exploring the volumes and their differences
